# 번역 가이드

## 규칙
### .ko.md
   docs.nestjs.com 레파지토리에서 content 하위에 있는 .md 파일들을 번역 후 .ko.md로 변경하시면 됩니다.
  
### Git
  번역 진행하려는 부분에 대해 이슈 등록하고 branch 따서 진행하시고 pull request 해주시면 됩니다.
  Pull request하실 때 Original 레파지토리에는 보내지 마세요.
